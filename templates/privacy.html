Política de Privacidade

Última atualização: 29 de junho de 2026

1. Dados Coletados

O Klaus Bot coleta os seguintes dados:

Dados do Usuário

  • ID do Discord — identificador único
  • Nome de usuário e nome de exibição
  • Avatar (URL)
  • Dados de perfil — background, border, itens comprados

Dados de Uso

  • Comandos utilizados
  • Estatísticas de jogo (vitórias, derrotas, streak)
  • Saldo de koins e transações
  • Nível de XP e progresso
  • Conquistas desbloqueadas

Dados do Servidor

  • ID do servidor
  • Nome do servidor
  • Configurações personalizadas (canal de boas-vindas, auto-mod, etc.)

2. Uso dos Dados

Os dados são utilizados para:

  • Fornecer as funcionalidades do Bot
  • Manter estatísticas e rankings
  • Melhorar a experiência do usuário
  • Detectar e prevenir abusos

3. Armazenamento

  • Dados são armazenados em servidores MongoDB seguros
  • Dados são retidos enquanto o Bot estiver ativo no servidor
  • Dados podem ser removidos sob solicitação

4. Compartilhamento

Não compartilhamos dados com:

  • Terceiros para fins de marketing
  • Anunciantes
  • Outros serviços

Dados podem ser compartilhados apenas quando:

  • Exigido por lei
  • Necessário para proteger direitos legais

5. Cookies e Rastreamento

O Bot não utiliza cookies. O dashboard usa cookies essenciais apenas para autenticação via OAuth2 do Discord.

6. Direitos do Usuário

Você tem direito a:

  • Acessar seus dados pessoais
  • Solicitar exclusão de seus dados
  • Exportar seus dados
  • Corrigir dados incorretos

7. Segurança

Implementamos medidas de segurança para proteger seus dados:

  • Conexões criptografadas (HTTPS)
  • Autenticação segura (OAuth2)
  • Acesso restrito a dados sensíveis

8. Menores de Idade

O Bot é destinado a usuários com 13+ anos (conforme Termos do Discord). Não coletamos dados intencionalmente de menores.

9. Alterações

Esta Política pode ser atualizada. Notificaremos sobre mudanças significativas no servidor de suporte.

10. Contato

Para exercer seus direitos ou esclarecer dúvidas:

  • Abra um ticket no dashboard do Bot
  • Entre em contato no servidor de suporte

Esta Política é aplicável apenas ao Klaus Bot e não representa afiliação com a Discord Inc.
